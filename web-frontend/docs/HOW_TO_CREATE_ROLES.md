## How to Assign Roles to Employees

### Option 1: From Team Page
1. Go to **"Team"** page (sidebar)
2. Find the employee
3. Click the purple **"Role"** button
4. Select the role from dropdown
5. Click **"Update Role"**

### Option 2: When Inviting New Employee
1. Go to **"Team"** page
2. Click **"Invite Employee"**
3. Fill in employee details
4. Select role from dropdown (coming soon in invite dialog)
5. Send invitation

## Tips & Best Practices

### ğŸ’¡ Role Naming
- Use clear, descriptive names
- Follow a consistent naming pattern
- Examples: "Sales Manager", "Support Agent Level 2"

### ğŸ’¡ Role Descriptions
- Briefly describe what this role can do
- Mention key responsibilities
- Example: "Manages sales team, approves deals over $10k, full CRM access"

### ğŸ’¡ Role Organization
- Create roles based on job functions
- Don't create too many similar roles
- Group by department or responsibility level

### ğŸ’¡ System Roles
- System roles are created by default
- They cannot be edited or deleted
